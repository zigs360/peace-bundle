# Peace Bundle

## 1. Overview
Peace Bundle is a Nigerian-based VTU and fintech platform that allows users to conveniently purchase mobile data, airtime, and utility services, while earning passive income through a structured referral and affiliate system.

## Platform Availability
- **Web Application**
- **Android Mobile Application**

## Brand Colors
- **Mint Green**: Primary (trust, growth, finance)
- **White**: Background and clarity
- **Gold**: Premium features, earnings, and highlights

## 2. User Authentication & Account Management
### 2.1 User Registration
- Full name, Email, Phone number, Password
- Optional referral code
- Email/SMS verification

### 2.2 User Login
- Email/Phone + Password
- Forgot password recovery

### 2.3 User Profile
- Update details
- Change PIN
- Security settings

## 3. Dashboard (Home Screen)
- Username, Account Level
- Wallet Balance (Gold highlight)
- Referral stats
- **Quick Actions**: Buy Data, Buy Airtime, Fund Wallet, Withdraw

## 4. Wallet System
- Funding via Virtual Account, Transfer, Payment Gateway (Paystack/Flutterwave)
- Real-time balance updates

## 5. VTU Services
- **Data**: MTN, Airtel, Glo, 9mobile
- **Airtime**: All networks
- **Utilities**: Electricity, Cable TV
- **History**: Transaction logs

## 6. Referral & Affiliate System
- Unique referral links
- Referral Bonus (1st funding)
- Ongoing Commissions
- Referral Dashboard

## 7. Packages & Account Levels
- **Smart Earner (Free)**: Basic access
- **Affiliate Package (Paid)**: Higher bonuses
- **Topuser / Reseller Package**: Best discounts, priority support

## 8. Withdrawals
